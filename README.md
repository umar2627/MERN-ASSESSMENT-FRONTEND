# MERN Assessment

This is a simple To-Do application built using the MERN stack (MongoDB, Express, React, Node.js).

## Available Scripts

npm start
npm run build

## Backend API

The backend API is expected to be running on `http://localhost:5000`. The following endpoints are used:

- `POST /addtodo` - Add a new to-do item
- `PUT /updatetodo/:id` - Update an existing to-do item
- `DELETE /deletetodo/:id` - Delete a to-do item
- `GET /fetchalltodos` - Fetch all to-do items

## Components

- **NavBar**: Displays the navigation bar.
- **InputFields**: Handles adding and editing to-do items.
- **ReadData**: Displays the list of to-do items.

## CSS Files

- **App.css**: Styles for the main App component.
- **frontend.css**: Styles for the input fields and data display.
